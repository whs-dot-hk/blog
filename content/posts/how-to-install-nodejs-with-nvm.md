You can install NodeJS with [Nvm][nvm] as follows

```sh
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
source ~/.bashrc
nvm install node
```

[nvm]: https://github.com/nvm-sh/nvm
